# 🌦 Weather Dashboard

A simple and interactive Weather Dashboard web application built using HTML, CSS, and JavaScript.  
It provides real-time weather updates and a 5-day forecast using the OpenWeather API.

---

## 🚀 Features

- Search weather by city name 🌍  
- Live current weather data 🌡  
- 5-day weather forecast 📅  
- Dynamic background changes based on weather conditions (Clear, Clouds, Rain, Snow, Thunder)  
- Clean and simple user interface  
- Easy navigation between pages  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeather API  

---

## 📁 Project Structure

Weather-Dashboard/
