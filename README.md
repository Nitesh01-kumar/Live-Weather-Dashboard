# 🌦 Weather Dashboard

A simple and interactive **Weather Dashboard web application** built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data using the **OpenWeather API** and displays current weather along with a 5-day forecast.

---

## 🚀 Features

- 🌍 Search weather by any city name
- 🌡 Real-time current weather data
- 📅 5-day weather forecast
- 🎨 Dynamic background changes based on weather conditions:
  - Clear ☀️
  - Clouds ☁️
  - Rain 🌧
  - Snow ❄️
  - Thunder ⚡
- 📱 Clean and responsive UI design
- 🧭 Simple navigation between pages

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeather API
